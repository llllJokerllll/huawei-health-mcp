# Contributing to Huawei Health MCP Server

Thank you for your interest in contributing! This is a new project and all help is welcome.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/huawei-health-mcp.git`
3. Create a branch: `git checkout -b feature/your-feature`
4. Make your changes
5. Commit: `git commit -m 'Add your feature'`
6. Push: `git push origin feature/your-feature`
7. Open a Pull Request

## Development Setup

See [docs/setup.md](docs/setup.md) for detailed setup instructions.

## Code Style

- **TypeScript**: Follow existing patterns, use strict mode
- **Python**: Follow PEP 8, use type hints
- **Kotlin**: Follow Android/Kotlin conventions

## Reporting Issues

Please use [GitHub Issues](https://github.com/llllJokerllll/huawei-health-mcp/issues) to report bugs or request features.
